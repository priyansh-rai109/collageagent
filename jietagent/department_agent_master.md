# Computer Science Engineering (CSE)

## Overview

Computer Science Engineering focuses on modern computing technologies including software development, artificial intelligence, cloud computing, and data science.

## Core Subjects

* Data Structures and Algorithms
* Operating Systems
* Database Management Systems
* Computer Networks
* Software Engineering
* Artificial Intelligence
* Machine Learning

## Laboratories

* Programming Lab
* Database Lab
* Networking Lab
* AI & ML Lab
* Software Development Lab

## Skills Developed

* Programming (Python, Java, C++)
* Problem Solving
* Web Development
* AI & Data Science
* System Design
